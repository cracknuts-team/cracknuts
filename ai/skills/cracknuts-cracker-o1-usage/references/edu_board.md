# EDU_EV_V1 教学开发板技术参考手册

## 1. 简介
**EDU_EV_V1** 是专为 **Cracker_O1** 控制板设计的配套教学实验板。它集成了工业级功率控制、多协议传感器以及丰富的交互外设，旨在帮助用户通过 Python SDK 或 AI Agent 快速掌握硬件控制逻辑。

---

## 2. 硬件功能分区
根据 `edu_board.md` 描述，电路板横向分为三个核心功能区域：

### 2.1 区域一：开关与电源区 (最左侧)
负责大电流负载及物理隔离控制实验。
* **继电器组 (Relay Group)**：
    * **硬件**：4路 **UB2-5NU** 继电器（U4, U6, U10, U14）。
    * **驱动**：**BTS7080-2EPA** 智能高侧开关，具备过流与过温保护。
* **CMOS 开关组**：
    * **硬件**：4路由 **DTC143Z** 数字晶体管控制的开关电路（Q1-Q4）。
* **电源接入**：12V DC 总接口（J33），为全板功率器件供电。

### 2.2 区域二：实验区域 1 (中间)
集成了电机驱动与多种通讯协议传感器。
* **动力模块**：
    * **步进电机接口** (J37)：支持四相步进电机控制。
    * **风扇控制** (Q5/FAN)：支持 PWM 调速。
    * **照明 LED** (Q6/LIGHT)：1W 大功率补光灯。
* **传感器模块**：
    * **温度测量 (SPI)**：**MAX31856**，支持 K 型热电偶。
    * **光照强度 (I2C)**：**BH1750**，测量环境照度。
    * **激光测距 (UART)**：**TOF10120** 接口。

### 2.3 区域三：实验区域 2 (最右侧)
用于基础 GPIO 逻辑及交互显示实验。
* **交互输入**：4路独立按键（BUTN1-4）。
* **LED 显示**：
    * **流水灯**：8 颗红光 LED (D8-D14 等)。
    * **红绿灯**：模拟十字路口东南西北三色信号。
    * **数码管**：**LD5161AS** 七段显示器。
* **高级显示**：8x8 LED 点阵，由三片 **74HC595** 移位寄存器级联驱动。

## 3. GPIO 逻辑映射 (SDK 调用参考)

基于 `cracker_o1_basic_usage` 规范，以下为开发时使用的 `pin_id` 映射：

### 3.1 数字输出 (Digital Output)
| 功能 | pin_id (丝印) | 逻辑说明 |
| :--- | :--- | :--- |
| **继电器 1-4** | `'relay1'` - `'relay4'` | 1 为吸合，0 为断开 |
| **流水灯 1-8** | `'led1'` - `'led8'` | 1 为点亮 |
| **红绿灯 (东)** | `'light_e_r/g/y'` | 红/绿/黄三色控制 |
| **数码管段选** | `'seg_a'` - `'seg_g'` | 段码控制 |

### 3.2 脉宽调制 (PWM)
| 功能 | pin_id (丝印) | 位号 | 说明 |
| :--- | :--- | :--- | :--- |
| **风扇转速** | `'fan_ctrl'` | Q5 | 频率推荐 1kHz-10kHz |
| **照明亮度** | `'light_ctrl'` | Q6 | 0-100 占空比调节 |

### 3.3 数字输入 (Digital Input)
| 功能 | pin_id (丝印) | 默认电平 | 模式设置 |
| :--- | :--- | :--- | :--- |
| **按键 1-4** | `'key1'` - `'key4'` | 1 (高) | 按下时返回 0 (低) |

*注：输入引脚需执行 `o1.digital_pin_mode(id, 1)`。*

## 4. 物理跳转连接 (Jumper Link) 逻辑
EDU 板设计了多个 **14pin 简牛座 (J35, J36, J39 等)**，允许通过物理排线改变信号流向：
1. **信号重定向**：可将主控的扩展 GPIO 跳转至任意 LED 或执行器。
2. **逻辑组合**：例如将 CMOS 开关的输出通过跳线接入流水灯，实现“软硬结合”的复杂电路控制教学。

## 5. 生产与制板参数
* **层数**：双层板 (Top/Bottom)。
* **尺寸**：127.0mm x 79.0mm。
* **厚度/材质**：1.6mm / FR-4 TG170。
* **工艺**：无铅喷锡，过孔塞油，绿色阻焊，白色字符。
