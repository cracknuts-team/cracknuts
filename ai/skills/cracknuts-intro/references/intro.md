# CrackNuts 简介（官方要点）

来源：`https://cracknuts.io/en/docs/0.18.x/introduction/`（抓取时间：2026-03-12）

## 项目定位

- CrackNuts 旨在提供“专业、易用、开源”的芯片安全分析解决方案。
- 能力覆盖：
  - 侧信道分析：功耗分析、电磁分析
  - 故障注入：时钟故障、低压故障
- 软件与硬件代码均计划开源，便于用户二次开发与集成。

## 组成与角色

- `CrackNuts`：上位机分析软件
  - 以 `Jupyter` 作为开发/使用环境
  - 提供设备与波形显示界面
  - 提供易用 API
  - 跨平台：Windows / Linux / macOS
- `Cracker`：测试设备（覆盖不同测试方向）
  - `Cracker-S1`：侧信道分析
  - `Cracker-G1`：故障分析（开发中）
  - `Cracker-F1`：硬件设计侧信道安全验证（开发中）
  - 后续设备按需求持续扩展
- `Nuts`：目标板/芯片
  - 示例：`Nut-stm32f103`、`Nut-stm32f4`
  - 支持定制目标板

## 部署特性（PoE）

- `Cracker` 设备支持 `PoE`（以太网供电）
- 优势：
  1. 一根网线完成供电与通信，部署更简洁
  2. 支持远程测试，降低物理距离限制
  3. 便于多设备联网，支持并行测试
